快速打包 Magsik 字体模块, 字体模板来自**落霞孤鹜 [lxgwshare]**.

## 使用方法

```shell
python mkm.py [-s] path_to_ttf_font(s)
```

如果你的字体压缩包或目录里有多种字体或字型, 请加```-s```参数
